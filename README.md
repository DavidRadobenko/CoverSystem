Key features:
=================================

Data generation:
- navmesh edge-walking
- 3D object scanning
    
Data persistence via octree

Navmesh error-handling
 
Multi-threaded, asynchronous cover generation

Real-time dynamic updates via Recast events

Profiler integration

Runs on UE 4.22 - 4.26
